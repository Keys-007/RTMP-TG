# RTMP TG
Play streams in your chats using this userbot !

# Requirements
> **Python:** `python3`  
> **FFmpeg:** `ffmpeg`  
> **Pyrogram:** `pyrogram`  
> **Youtube Dl :** `youtube_dl` - (optional)

# How to run?
> run `pip install -r requirements.txt`  
> fill your vars in `local.env`
> run `python3 main.py`  
> Now play using `!stream (url)`
> Njoy!
