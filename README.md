# RTMP TG
Play streams in your chats using this userbot !

# Requirements
> **Python:** `python3`  
> **FFmpeg:** `ffmpeg`  
> **Pyrogram:** `pyrogram`  

# How to run?
> run `pip install -r requirements.txt`  
> fill your vars in `local.env`
> run `python3 main.py`  
> Now test your userbot with `!ping`
